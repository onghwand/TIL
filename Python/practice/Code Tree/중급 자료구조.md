> hashmap 기본

```python
d = {}
n = int(input())
for i in range(n):
    op, *nums = input().split()
    if op == 'add':
        d[nums[0]] = nums[1]
    elif op == 'find':
        if nums[0] in d:
            print(d[nums[0]])
        else:
            print('None')
    elif op == 'remove':
        d.pop(nums[0])
```

## TreeMap

- SortedDict는 key값을 기준으로 정렬된 hashmap(dict)이다. 

> treemap 기본

```python
from sortedcontainers import SortedDict

sd = SortedDict()
n = int(input())
for i in range(n):
    op, *nums = input().split()
    nums = list(map(int, nums))
    if op == 'add':
        sd[nums[0]] = nums[1]
    elif op == 'find':
        if nums[0] in sd:
            print(sd[nums[0]])
        else:
            print('None')
    elif op == 'remove':
        sd.pop(nums[0])
    elif op == 'print_list':
        if sd:
            for k in sd.keys():
                print(sd[k], end=' ')
            print()
        else:
            print('None')
```

## TreeSet

- SortedSet은 정렬된 hashSet이다. 
- bisect_left는 입력값을 기준으로 크거나 작은 값의 인덱스, bisect_right는 큰 값의 인덱스를 반환한다. 

> treeset 기본

```python
from sortedcontainers import SortedSet

s = SortedSet()  
n = int(input())

for i in range(n):
    op, *nums = input().split()
    nums = list(map(int, nums))
    if op == 'add':
        s.add(nums[0])
    elif op == 'remove':
        s.remove(nums[0])
    elif op == 'find':
        if nums[0] in s:
            print('true')
        else:
            print('false')
    elif op == 'lower_bound':
        print(s[s.bisect_left(nums[0])] if s.bisect_left(nums[0]) < len(s) else 'None')
    elif op == 'upper_bound':
        print(s[s.bisect_right(nums[0])] if s.bisect_right(nums[0]) < len(s) else 'None')
    elif op == 'largest':
        print(s[-1] if s else 'None')
    elif op == 'smallest':
        print(s[0] if s else 'None')
```

> 가까운 숫자

```python
from sortedcontainers import SortedSet

n = int(input())
arr = list(map(int, input().split()))
s = SortedSet([0])

minV = 1000000000
for i in range(n):
    num = arr[i]
    s.add(num)
    r_idx = s.bisect_left(num) + 1
    l_idx = s.bisect_left(num) - 1
    if r_idx < len(s) and l_idx >= 0:
        tmp = min(abs(num-s[r_idx]), abs(num-s[l_idx]))
    elif r_idx < len(s):
        tmp = abs(num-s[r_idx])
    else:
        tmp = abs(num-s[l_idx])
    if tmp < minV:
        minV = tmp
    print(minV)
```
