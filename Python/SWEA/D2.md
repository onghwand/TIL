## 2001. 파리 퇴치

```python
T = int(input())

for test_case in range(1, T + 1):
    N, M = map(int, input().split())
    # 2차원 배열 생성
    tile = []
    for i in range(N):
        tile.append(list(map(int, input().split())))
        
    # 인접한 4개의 합 max구하기
    maximum = 0
    # (i,j) 선택 
    for i in range(N-M+1):
        for j in range(N-M+1):
            flies = 0
            for k in range(M):
                for l in range(M):
                    flies += tile[i+k][j+l]
                
            # max 갱신        
            if maximum < flies:
                maximum = flies
    
        
    print(f'#{test_case} {maximum}')
```

## 1983. 조교의 성적 매기기

```python
import math
T = int(input())
for test_case in range(1, T + 1):
    nums, target = map(int, input().split())
    totals = []
    for num in range(nums):
        scores = list(map(int, input().split()))
        scores[0] *= 0.35
        scores[1] *= 0.45
        scores[2] *= 0.2
        totals.append(sum(scores))
        
    grade = {1:'D0',
             2:'C-',
             3:'C0',
             4:'C+',
             5:'B-',
             6:'B0',
             7:'B+',
             8:'A-',
             9:'A0',
             10:'A+'}
    
    target_percent_rank = (sorted(totals).index(totals[target-1]) + 1) / len(totals)
   
    result = grade[math.ceil(target_percent_rank*10)]

    print(f'#{test_case} {result}')
```

## 1979. 어디에 단어가 들어갈 수 있을까

```python
T = int(input())
for test_case in range(1, T + 1):
    N, K = map(int, input().split())
    
    tile = []
    for i in range(N):
        tile.append(list(input().split()))
        
    pieces = []
    #가로
    for i in range(N):
        pieces.extend((''.join(tile[i])).split('0'))
    #세로
    for i in range(N):
        zero_one = ''
        for j in range(N):
            zero_one += tile[j][i]
        pieces.extend(zero_one.split('0'))
        
    cnt = pieces.count('1'*K)
       
    print(f'#{test_case} {cnt}')
```

## 1970. 쉬운 거스름돈

```python
T = int(input())
for test_case in range(1, T + 1):
    print(f'#{test_case}')
    # 받은 돈
    money = int(input())
    # 지폐 종류, 개수 0으로 초기값
    bill = {50000:0,
            10000:0, 
            5000:0, 
            1000:0, 
            500:0, 
            100:0, 
            50:0, 
            10:0}
    # 단위가 큰 지폐부터 개수 할당
    for key in bill.keys():
        bill[key] = money // key
        money = money % key  
    # 답안 형식 맞추기
    cnts = map(str, list(bill.values()))
    print(" ".join(cnts))
```

## 1959. 두 개의 숫자열

```python
T = int(input())
for test_case in range(1, T + 1):
    A, B = map(int, input().split())
    numA = list(map(int, input().split()))
    numB = list(map(int, input().split()))
    total = 0
    if A <= B:
        for i in range(B-A+1): 
            subtotal = 0
            for a,b in zip(numB[i:i+A], numA):
                subtotal += a*b
            if total < subtotal:
                total = subtotal
    else:
        for i in range(A-B+1):
            subtotal = 0
            for a,b in zip(numA[i:i+B], numB):
                subtotal += a*b
            if total < subtotal:
                total =subtotal
    print(f'#{test_case} {total}')
```

## 1948. 날짜 계산기

```python
T = int(input())
for test_case in range(1, T + 1):
    result = 0
    month_day={1:31,
               2:28,
               3:31,
               4:30,
               5:31,
               6:30,
               7:31,
               8:31,
               9:30,
               10:31,
               11:30,
               12:31}
    
    start_month, start_day, end_month, end_day = map(int,input().split())
    if start_month == end_month:
        result += end_day - start_day +1
    else:
        for month in range(start_month, end_month):
            result += month_day[month]
        result = result - start_day + end_day + 1 
    
    print(f'#{test_case} {result}')
```
