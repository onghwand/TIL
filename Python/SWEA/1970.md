## 1970. 쉬운 거스름돈

```python
T = int(input())
for test_case in range(1, T + 1):
    print(f'#{test_case}')
    # 받은 돈
    money = int(input())
    # 지폐 종류, 개수 0으로 초기값
    bill = {50000:0,
            10000:0, 
            5000:0, 
            1000:0, 
            500:0, 
            100:0, 
            50:0, 
            10:0}
    # 단위가 큰 지폐부터 개수 할당
    for key in bill.keys():
        bill[key] = money // key
        money = money % key  
    # 답안 형식 맞추기
    cnts = map(str, list(bill.values()))
    print(" ".join(cnts))
```
