## 3431. 준환이의 운동관리

```python
T = int(input())
for tc in range(1, T+1):
    L, U, X = map(int, input().split())
    if L <= X <= U:
        ans = 0
    elif X < L:
        ans = L - X
    else:
        ans = -1
    print(f'#{tc} {ans}')
```

## 3376. 파도반 수열

```python
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    arr = [1,1,1,2,2]
    i = 4
    while len(arr) <= N:
        arr.append(arr[i]+arr[i-4]) #4번째 전 길이랑 합쳐져서 다음 것이 만들어짐
        i += 1

    print(f'#{tc}',arr[N-1])
```

## 3314. 보충학습과 평균

```python
T = int(input())
for tc in range(1, T+1):
    arr = list(map(int, input().split()))
    total = 0
    for score in arr:
        if score < 40:
            total += 40
        else:
            total += score
    print(f'#{tc} {total//len(arr)}') # 문제조건에 모든 점수가 5의배수라고 해서 몫으로
```
