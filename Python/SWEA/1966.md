## 1966. 숫자를 정렬하자

> 정답

```python
T = int(input())

for t in range(1, T+1) :
    N = int(input())
    num = list(map(int, input().split()))
    num.sort()

    print("#{}".format(t), end=' ')
    for i in range(N) :
        print(num[i], end=' ')
    print()
```

> 내 오답

- 왜 오답인지 아직도 모르겠다..

```python
T = int(input())
for test_case in range(1, T + 1):
    cnt = int(input())
    if cnt >= 5 and cnt <=50: #제약조건
        numbers = list(input().split())
    numbers = sorted(numbers)
    result = ' '.join(numbers)
    
    print(f'#{test_case} {result}')
```
